訂購方式
我們提供網路訂購、電話訂購、傳真訂購三種方式說明如下：
01網路訂購
最佳下單方式-電腦下單 ( 手機下單容易造成失敗或無法完成付款 )
為了確保產品品質，我們將宅配包裝箱尺寸統一為長25.5公分X寬23公分X高12公分
訂購總金額滿3000元單件單一溫層免運費
未滿3000元時各溫層收費如下
常溫運費為150元 ( 乾貨、罐頭、調味料、及常溫相關產品 )
冷藏運費為150元 ( 蔬菜、冷藏肉品及須冷藏相關產品 )
冷凍運費為150元 ( 牛排類、冷凍肉品、急凍海鮮及冷凍相關產品 )
因不同溫層無法同捆配送，須以實際商品件數計算運費。
網路訂購請依照網站購物指示，完成訂購後將會有專人與您確認，謝謝!!
02電話訂購
歡迎撥打訂購專線：0965-108-581會有專人為您服務
03傳真訂購
請將訂單填妥後傳真至：02-2341-1399收件後會有專人與您確認，亦可來電確認訂單

04網站付款方式
網站付款方式可選擇”ATM轉帳””信用卡付款”或”貨到付款”
ATM轉帳：請在下單後24小時內完成付款，逾時系統訂單會自動失效。
貨到付款：選擇貨到付款方式請您將服務人員報價之金額直接付款給送貨人員即可 ( 手續費由本公司吸收 )

【商品退換貨、使用及保存須知】

本公司商品為【生鮮、調理食品】，不符合七日鑑賞期商品，請您於下單前確認以下內容，以避免消費爭議，謝謝。

  • 在您收到貨品後，請當下檢查您所訂購的商品，如有非人為因素所致之商品損毀、或運輸過程造成包裝破損不完整或失溫導致食品變質，煩請您在收到貨品的半小時內拍照存證後來電或來信告知並將商品冷凍保存，我們會進行商品瑕疵或損壞鑑定，並儘速為您更換新品。
  • 下列情形可能影響您的退貨權限：
    ＊在不影響您檢查商品情形下，將真空包裝拆開。
    ＊在您收到商品後，無立即將商品冷凍保存。
    ＊其他逾越檢查之必要或可歸責於您的事由，致商品有毀損、滅失或變更者。
  • 若您已取得紙本發票，請於退貨時一併退還。

  • 請您以送貨廠商使用之包裝盒將退貨商品包裝妥當若原包裝盒已遺失，請另使用其他箱子裝箱。

  • 當您申請退換貨後，請主動向貨運人員索取單據，並保留至退換貨完成，以利日後查詢。
